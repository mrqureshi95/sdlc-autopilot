# Evaluation Fixtures

These fixtures are intentionally small, imperfect, and in some cases intentionally insecure.

Purpose:

- give AI agents realistic bug-fix and feature tasks
- include planted defects and vulnerability patterns
- measure whether the skill drives better engineering behavior

Important:

- These fixtures are not production code.
- Some fixtures intentionally contain SQL injection, XSS, missing validation, or other insecure patterns.
- Security findings inside this directory are often expected by design.
- Do not deploy code from this directory.
